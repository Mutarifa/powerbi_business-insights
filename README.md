
# Business Insight 360 Dashboard

##  Overview
This project is an end-to-end Business Intelligence dashboard built in Power BI to analyze business performance across Finance, Sales, Marketing, and Supply Chain.

The dashboard provides a centralized view of key metrics, enabling better understanding of profitability, performance trends, and forecasting accuracy.

---

## Dataset
- Source: Codebasics
- Size: ~1 million rows
- Includes data related to sales, costs, customers, products, and forecasts

---

## Data Preparation & Modeling
- Cleaned and transformed data using Power Query
  - Removed duplicates
  - Handled inconsistencies
- Created calculated columns where required
- Built DAX measures for optimized performance and reduced model size

---

##  Dashboard Views

### 1. Finance View
- Dynamic Profit & Loss (P&L) statement
- Revenue, cost, and profit analysis
- Time-based trends (monthly, quarterly, yearly)
- Interactive filtering across visuals

---

### 2. Sales View
- Customer performance (Net Sales, Gross Margin, GM%)
- Product performance analysis
- Scatter plot to identify high-value customers
- Unit economics:
  - Pre vs Post invoice deductions
  - COGS vs Gross Margin

---

### 3. Marketing View
- Net Profit and Net Profit %
- Product and customer profitability analysis
- Performance matrix for identifying truly profitable segments
- Revenue to profit conversion insights

---

### 4. Supply Chain View
- Forecast accuracy tracking (current vs last year)
- Net error and net error %
- Trend analysis over time
- Customer and product-level forecasting performance

---

##  Key Metrics
- Net Sales
- Gross Margin & Gross Margin %
- Net Profit & Net Profit %
- Forecast Accuracy %
- Net Error & Net Error %

---

##  Key Insights
- High revenue does not always translate to high profitability
- Product-level margins vary significantly
- Some customers contribute more to revenue but less to profit
- Forecast accuracy varies across products and customers, impacting planning
